<task type="auto">
  <name>Task 1: Feed server actions</name>
  <files>src/app/actions/feeds.ts</files>
  <action>
Create `src/app/actions/feeds.ts` with 'use server' directive containing these server actions:

1. **addFeed(brandId: number, url: string, type: 'rss' | 'youtube' | 'reddit' | 'google_news'): Promise<{ id?: number; error?: string }>**
   - Validate URL is non-empty and starts with http:// or https://
   - Auto-detect type from URL if possible:
     - Contains 'youtube.com/feeds/videos.xml' -> 'youtube'
     - Contains 'reddit.com/r/' and ends with '.rss' -> 'reddit'
     - Contains 'news.google.com/rss' -> 'google_news'
     - Otherwise -> use provided type
   - Insert into feedSources with defaults: pollInterval=5, relevanceThreshold=6, targetPlatforms=null (means all), enabled=1
   - revalidatePath(`/brands/${brandId}/feeds`)
   - Return { id: result.id }

2. **updateFeed(feedId: number, updates: { pollInterval?: number; relevanceThreshold?: number; targetPlatforms?: string[]; enabled?: number }): Promise<{ error?: string }>**
   - Update feedSources row with provided fields
   - Lookup brandId for revalidation
   - revalidatePath(`/brands/${brandId}/feeds`)

3. **deleteFeed(feedId: number): Promise<{ error?: string }>**
   - Lookup brandId first for revalidation
   - Delete all feedEntries for this feedSourceId first (cascading manually)
   - Delete the feedSources row
   - revalidatePath(`/brands/${brandId}/feeds`)

4. **getBrandFeeds(brandId: number): Promise<Array<FeedWithStats>>**
   - Query feedSources for the brand
   - For each feed, count total feedEntries, count relevant entries (relevanceScore >= threshold), count processed entries
   - Return array of feed objects with stats: { ...feed, totalEntries: number, relevantEntries: number, processedEntries: number }
   - Define `FeedWithStats` type and export it

5. **updateAutomationLevel(brandId: number, level: 'manual' | 'semi' | 'mostly' | 'full'): Promise<{ error?: string }>**
   - Update brands.automationLevel for the brand
   - revalidatePath(`/brands/${brandId}/feeds`)
  </action>
