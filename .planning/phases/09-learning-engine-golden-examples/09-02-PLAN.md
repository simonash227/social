**Step 2: Modify buildSystemPrompt signature and body**

Change from `function buildSystemPrompt(brand: BrandRow): string` to:
```typescript
function buildSystemPrompt(
  brand: BrandRow,
  learnings?: BrandLearning[],
  goldenExamples?: GoldenExample[]
): string
```

In the function body, BEFORE the closing two lines (`parts.push('')` and `parts.push('Respond with ONLY valid JSON...')`), inject golden examples and learnings:

```typescript
// After the existing examplePosts block and before closing instruction:

// Inject golden examples as style references
if (goldenExamples && goldenExamples.length > 0) {
  parts.push('')
  parts.push('TOP PERFORMING POSTS — USE AS STYLE REFERENCE (do not copy, generalize the pattern):')
  for (const ex of goldenExamples) {
    parts.push(`[${ex.platform.toUpperCase()} — engagement score ${ex.engagementScore}]`)
    parts.push(ex.content.slice(0, 300)) // cap at 300 chars to manage token budget
    parts.push('---')
  }
}

// Inject approved learnings
if (learnings && learnings.length > 0) {
  parts.push('')
  parts.push('LEARNINGS FROM YOUR TOP PERFORMERS (apply these):')
  for (const l of learnings) {
    const prefix = l.type === 'avoid_pattern' ? 'AVOID' : 'DO'
    parts.push(`- [${prefix}] ${l.description}`)
  }
}
```

The closing JSON instruction lines remain AFTER these blocks (unchanged).

Do NOT change the runRewrite signature or add complexity -- the current `buildSystemPrompt(brand)` call in runRewrite is fine as-is since learnings are optional and backward-compatible.

Import: `getDb` from `@/db`, `brandLearnings`, `posts` from `@/db/schema`, `eq` from `drizzle-orm`, `revalidatePath` from `next/cache`.

Implement these server actions:

**approveLearning(learningId: number): Promise<{ error?: string }>**
- Update `brandLearnings` SET `status = 'approved'`, `isActive = 1`, `updatedAt = now`.
- `revalidatePath('/brands/[id]/learnings', 'page')`.

**rejectLearning(learningId: number): Promise<{ error?: string }>**
- Update `brandLearnings` SET `status = 'rejected'`, `isActive = 0`, `updatedAt = now`.
- `revalidatePath('/brands/[id]/learnings', 'page')`.

**toggleLearning(learningId: number, isActive: boolean): Promise<{ error?: string }>**
- Update `brandLearnings` SET `isActive = isActive ? 1 : 0`, `updatedAt = now`.
- `revalidatePath('/brands/[id]/learnings', 'page')`.

**runManualAnalysis(brandId: number): Promise<{ error?: string }>**
- Dynamic import `analyzeAllPlatformsForBrand` from `@/lib/learning-engine`.
- Call with `{ force: true }` to bypass 7-day gate.
- `revalidatePath('/brands/' + brandId + '/learnings')`.
- Wrap in try/catch; return `{ error: message }` on failure.

**pinGoldenExample(postId: number, brandId: number): Promise<{ error?: string }>**
- Update `posts` SET `isGoldenPinned = 1` WHERE `id = postId`.
- `revalidatePath('/brands/' + brandId + '/golden-examples')`.

**unpinGoldenExample(postId: number, brandId: number): Promise<{ error?: string }>**
- Update `posts` SET `isGoldenPinned = 0` WHERE `id = postId`.
- `revalidatePath('/brands/' + brandId + '/golden-examples')`.
