   <div className="flex flex-wrap items-center gap-3">

      - Value: `sortOrder`

      - Options: "Newest first" (value="newest"), "Oldest first" (value="oldest")

   b) Date range select (width w-[150px]):
      - Value: `dateRange`
      - onValueChange: `(v) => setDateRange((v ?? 'all') as 'all' | 'week' | 'month')`
      - Options: "All dates" (value="all"), "This week" (value="week"), "This month" (value="month")

      - Value: `typeFilter`
      - onValueChange: `(v) => setTypeFilter((v ?? 'all') as 'all' | 'generated')`
      - Options: "All types" (value="all"), "Generated" (value="generated")

   After the selects, add a results count:

     {filteredImages.length} of {images.length} images
   </span>

6. Add a "no results" state inside the grid area: if `filteredImages.length === 0 && images.length > 0`, show a centered message "No images match the current filters." with a Button to reset filters (sets all three states back to defaults).
