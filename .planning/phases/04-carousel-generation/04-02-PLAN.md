**1. generateSlideContent action:**
```typescript
export async function generateSlideContent(
  brandId: number,
  sourceText: string,
  slideCount: number,
): Promise<{ slides: Array<{ title: string; body: string }>; costUsd: number; error?: string }>
```

Implementation:
- Check AI spend limit (checkAiSpend). Return error if over limit.
- Look up brand from DB (get brand row for niche, voiceTone context)
- Get model config (getModelConfig().primary)
- Build prompt instructing AI to create exactly slideCount slides:
  - Slide 1 = Hook: attention-grabbing title, 5-8 words, no body needed
  - Slides 2 to (N-1) = Content: each covers one key insight. Title: short label (2-5 words). Body: 1-2 sentences max.
  - Slide N = CTA: summarize value + call to action. Title = summary phrase. Body = CTA text.
  - Include brand voice and niche in prompt for brand-aware content
  - Request JSON response: `{ "slides": [{ "title": "...", "body": "..." }, ...] }`
- Call Anthropic via getBreaker('anthropic').call(...)
- Parse response with parseJsonResponse
- **Validate slide count**: if slides.length !== slideCount, pad with empty { title: '', body: '' } or trim. Log warning on mismatch.
- Log AI spend
- Return { slides, costUsd }

**2. renderAndSaveCarousel action:**
```typescript
export async function renderAndSaveCarousel(params: {
  brandId: number
  templateId: string
  sourceText: string
  slides: Array<{ title: string; body: string }>
}): Promise<{ carouselId: number; error?: string }>
```

Implementation:
- Look up brand for primaryColor, secondaryColor, logoUrl, ctaText
- Look up first connected social account username for the handle (query socialAccounts where brandId, status='connected', limit 1). Fallback: use brand.name if no accounts connected.
- Build BrandStyle object: { primaryColor: brand.primaryColor ?? '#6366f1', secondaryColor: brand.secondaryColor, logoUrl: brand.logoUrl, fontFamily: 'Inter' }
- Map slides array to SlideData[]:
  - Index 0: type='hook', title=slides[0].title, body=slides[0].body (may be empty for hook)
  - Last index: type='cta', title=slides[last].title, body=slides[last].body, ctaText=brand.ctaText ?? 'Follow for more', handle=`@${account.username}` or `@${brand.name}`
  - Middle indices: type='content'
  - All slides get slideNumber (1-based) and totalSlides
- Validate templateId is in TEMPLATE_IDS. Default to 'minimal' if invalid.
- Call renderCarouselSlides({ templateId as TemplateId, brandId, brandStyle, slides: slideDataArray })
- Insert into carousels table: { brandId, templateId, sourceText, slideCount: slides.length, status: 'ready' }
- Insert into carouselSlides table: one row per slide with { carouselId, slideIndex, title, body, r2Key, thumbKey }
- Return { carouselId }
- Wrap entire function in try/catch, return { carouselId: 0, error: message } on failure
